List of HTML Tags

  1. <html>:Html tag, it is the root element of an HTML document.
  2. <head>:Head tag, it Contains meta-information about the document.
  3. <title>:Title tag, it Sets the document's title.
  4. <script>:Script tag, it Embeds or references JavaScript code.
  5. <style>:Style tag, it Contains CSS code or links to stylesheets.
  6. <p> :Paragraph tag, Represents a paragraph of text.
  7. <a> :Anchor tag, it Creates hyperlinks.
  8. <img>:Image tag, it Embeds images.
  9. <ul>:Unorder list tag, it Defines an unordered list.
  10. <ol>:Order list tag, It Defines an ordered list.
  11. <li>:List item tag, it Represents list items.
  12. <div>:Div tag, it is a generic container element.
